Что такое абстрактный класс в Java?
Абстрактный класс - это класс, который не может быть создан напрямую, а только наследован другими классами. Он может содержать как реализацию методов, так и абстрактные методы, которые должны быть реализованы в подклассах
Что такое интерфейс в Java?
Интерфейс - это абстрактный тип данных, который определяет набор методов, которые должны быть реализованы классами, которые реализуют этот интерфейс. Интерфейсы могут содержать только абстрактные методы, константы и методы по умолчанию.
Что такое полиморфизм в Java?
Полиморфизм - это концепция объектно-ориентированного программирования, которая позволяет объектам разных классов использовать одинаковые методы. Это достигается благодаря наследованию и интерфейсам.
Что такое исключение в Java?
Исключение - это объект, который представляет ошибку или исключительную ситуацию, возникшую во время выполнения программы. Исключения могут быть обработаны с помощью блока try-catch или переданы вызывающему методу.
Что такое коллекции в Java?
Коллекции - это объекты, которые используются для хранения и манипулирования группами элементов. В Java есть множество различных типов коллекций, таких как списки, множества и карты.
Что такое лямбда-выражения в Java?
Лямбда-выражения - это способ определения анонимных функций в Java. Они позволяют передавать функции как параметры методов и использовать их в качестве значений переменных.
Что такое дженерики в Java?
Дженерики - это механизм, который позволяет создавать обобщенные типы данных, которые могут работать с различными типами данных.
Какие есть приемущества и недостатки у дженериков в Java?
Использование дженериков в Java позволяет создавать более безопасный и читаемый код, а также повышает производительность программы. Дженерики также позволяют программистам избежать ошибок типизации, которые могут возникнуть при работе с необобщенными типами данных. Одним из недостатков дженериков в Java является то, что они могут приводить к увеличению размера байт-кода программы. Кроме того, использование дженериков может быть сложным для новых программистов, которые не знакомы с концепцией обобщенных типов данных
Как объявить обобщенный класс в Java?
Для объявления обобщенного класса в Java нужно указать параметр типа в угловых скобках после имени класса. Например: public class Class<T> { }.
Как объявить обобщенный метод в Java?
Для объявления обобщенного метода в Java нужно указать параметр типа в угловых скобках перед возвращаемым типом. Например: public <T> void method(T arg) { }.
Какие ограничения можно наложить на параметры типа в Java?
В Java можно наложить ограничения на параметры типа с помощью ключевого слова extends. Например: public class MyClass<T extends Number> { }.
